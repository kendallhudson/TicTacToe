TIC-TAC-TOE!

Instructions

  1. Input Player 1 name and select "Add Player 1" button
  2. Input Player 2 name and select "Add Player 2" button OR just select the button to play against the Computer
  3. Take turns marking each square once to place an "X" or an "O"
  4. Try to get 3 in a row before the other player does
  5. Have fun and don't forget to play again when you're done!!
